Чтобы протестировать ЭТО необходимо:
1. Запустить скрипт start.bat
2. Засунуть готовый WAR фаил из папки target в папку apache-tomcat\webapps
3. Запустить TomCat ( зайти в папку apache-tomcat\bin и запустить скрипт startup ) 
4. Ввести в адрессной строке браузера localhost
5. Войти в систему и запустить развернутое веб-приложение

Алтернативный путь запуска

1. Открыть проект через IDEA
2. Нажать кнопку edit configuration и добавить TomCat localhost 
3. Там снизу будет fix, нужно нажать на нее, и в application context написать /MVC-spring
4. Сохранить настройки и запустить приложение

Необходимые условия, должна быть создана пустая база данных test.
